# MeuPortfolio - Playwright - Demo

### Objetivo

O objetivo é demonstrar o uso da ferramenta playwright para extrair dados de um website sejam eles: screenshots, conteúdo html e até mesmo retornos Json!!
O playwright é uma ferramenta criada pela Microsoft e pode ser utilizada em Python, Node, C# e Go. Você pode encontrar maiores detalhes do Playwright [aqui](https://playwright.dev/).
Não deixa de dar uma estrela a esse super projeto [aqui](https://github.com/microsoft/playwright)

### Projeto

O projeto foi desenvolvido em .net 6 e (Getting Started)[https://playwright.dev/dotnet/docs/intro/] do próprio site para instalação do playwright na minha máquina.

Usei como exemplo uma página de analise de ativos da empresa (Meuportfolio)[https://meuportfol.io/analise-de-ativos/fundos/capitania-premium-mst-fi-rf-cred-priv-lp] qual eu trabalho.
Com essa url o playwright foi capaz de, com poucas linhas de código, tirar um print de tela, o conteúdo HTML e todo as requisições XHR (chamadas APIs)!! Isso é insano!
